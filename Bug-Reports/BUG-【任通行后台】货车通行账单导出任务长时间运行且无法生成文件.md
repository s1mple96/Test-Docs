# BUG-【任通行后台】货车通行账单导出任务长时间运行且无法生成文件

## 1. BUG描述

在导出中心对货车用户通行账单发起导出任务后，任务持续执行近 2 小时，执行成功数量持续暴涨至千万级（实测超 1500 万条），始终无法生成导出文件，下载链接为空。

**影响**：
- ✅ 提交导出任务可以成功（返回任务 ID）
- ✅ 任务状态更新正常
- ❌ 货车通行账单导出任务长时间无法结束
- ❌ 导出成功数量异常暴涨，与实际账单数据量严重不符
- ❌ 导出记录始终无文件名（`export_name=NULL`）和下载链接（`export_url=NULL`）
- ❌ 用户无法下载导出文件

---

## 2. 复现步骤

### 前置条件
- 测试环境：`192.168.1.60`
- 系统：任通行后台
- 数据库：`rtx`
- 测试账号：admin
- 测试数据：身份证 `441622199602075195`，钱包 ID `7b4bfaa4a8de11f084f1141877512abb`

### 操作步骤

**Step 1**：在导出中心输入身份证查询用户，选择车辆类型为「货车」，导出类型选「通行账单」，不选择具体车牌（即导出全部车辆），不填写时间范围，点击「导出」按钮

```bash
curl -X POST 'http://788360p9o5.yicp.fun/rtx-admin/center/export/createTask' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzUxMiJ9...' \
  -H 'Content-Type: application/json;charset=UTF-8' \
  -d '{
    "carType": "2",
    "idCard": "441622199602075195",
    "exportTypes": ["1"],
    "carNums": [],
    "walletId": "7b4bfaa4a8de11f084f1141877512abb"
  }'
```

**Step 2**：等待约 10～30 分钟后，在导出记录列表查看任务进度
