# BUG-【任货行后台】流量卡申办日志页面首次无查询按钮

## 1. BUG描述

使用admin账号登录任货行管理后台，首次进入"流量卡申办日志"页面时，查询按钮不显示。但切换到其他页面（如设备注销页面）后，再重新打开"流量卡申办日志"页面，查询按钮就正常显示了。

**影响**：
- ✅ 其他管理页面的查询按钮正常显示（如设备注销页面）
- ❌ 流量卡申办日志页面首次访问时查询按钮不显示
- ✅ 重新打开流量卡申办日志页面后查询按钮正常显示
- ✅ admin账号拥有所有权限，理论上应该能看到所有按钮

---

## 2. 复现步骤

### 前置条件
- 测试环境: 192.168.1.60
- 系统: 任货行管理后台
- 数据库: hcb
- 测试账号: admin
- 浏览器: Chrome 131

### 操作步骤

**Step 1**: 清除浏览器缓存，使用admin账号登录任货行管理后台

```
用户名: admin
密码: [admin密码]
```

**Step 2**: 进入"业务管理" → "流量卡申办日志"页面

访问URL: `http://192.168.1.60:8080/sim/applyLog/list.do`

**Step 3**: 观察页面顶部查询区域

查看是否显示查询按钮（搜索图标按钮）：
```
[车牌号输入框] [开始日期] [结束日期] [?查询按钮?]
```

**Step 4**: 切换到其他管理页面（如设备注销页面）

进入"业务管理" → "设备注销管理"

**Step 5**: 观察设备注销页面的查询按钮

验证其他页面的查询按钮是否正常显示。

**Step 6**: 关闭"流量卡申办日志"的Tab页

点击Tab页的关闭按钮 ×

**Step 7**: 重新打开"流量卡申办日志"页面

再次进入"业务管理" → "流量卡申办日志"

**Step 8**: 再次观察查询按钮

查看查询按钮是否显示。
