# 【任货行】1017503 非湖南江苏先享月费1-2月份数据初始化测试用例

> 版本：V1.0（2026-03-30）  
> 需求：任货行非湖南、江苏的先享月费1-2月份数据初始化  
> 关联提交：68f68c14

---

## 需求说明

**Step 1 - 插入全量钱包账单（hcb_complete_wallet_bill）**

- 数据来源：非湖南、非江苏运营商（LTK/HTK/JTK/MTK/CTK）的1-2月份月费账单（bill_type=1）
- 不管原记录扣款成功还是失败，有一条记录则插入一条
- 插入：`bill_type=3`（月管理费账单2），`bill_amount=15元`

**Step 2 - 手动生成先享月费账单（hcb_monthly_paylater_wallet_bill）**

- 基于 Step 1 插入的数据，生成1-2月份的先享月费账单

**Step 3 - 4月1日前执行扣款**

**触发接口**：
```bash
curl "http://127.0.0.1:8082/hcbTimer-business/test/manualGenerateAndDeductBills.do"
```

**运营商范围**：LTK、HTK、JTK、MTK、CTK（非湖南 TXB、非江苏 XTK）

---

## 1. Step 1 验证：全量钱包账单插入

### 1.1 数据插入正确性

#### TC-001 执行接口后-非湖南非江苏1月份有月费账单的用户生成 bill_type=3 记录
- **前置条件**:
  - 系统: hcbTimer-business
  - 数据库: hcb
  - `hcb_complete_wallet_bill` 中存在 operator_code IN ('LTK','HTK','JTK','MTK','CTK')、bill_type=1、bill_date BETWEEN 20260101 AND 20260131 的数据
  - 测试环境: 192.168.1.60（本机执行接口）
- **测试步骤**:
  1. 执行前记录基线：
     ```sql
     SELECT operator_code, COUNT(*) as cnt
     FROM hcb_complete_wallet_bill
     WHERE bill_type = 3 AND operator_code IN ('LTK','HTK','JTK','MTK','CTK')
     AND bill_date = 20260101;
     ```
  2. 执行接口：
     ```bash
     curl "http://127.0.0.1:8082/hcbTimer-business/test/manualGenerateAndDeductBills.do"
     ```
  3. 执行后查询新生成记录：
     ```sql
     SELECT user_id, operator_code, bill_type, bill_amount, actual_amount,
            fail_amount, bill_status, bill_date, remark
     FROM hcb_complete_wallet_bill
     WHERE bill_type = 3 AND operator_code IN ('LTK','HTK','JTK','MTK','CTK')
     AND bill_date = 20260101
     ORDER BY create_time DESC LIMIT 20;
     ```
- **预期结果**:
  - 有1月份月费账单的用户，各生成一条 bill_type=3 记录
  - `bill_date = 20260101`（固定1月1日，非今日日期）
  - `bill_amount = 15.00`
  - `actual_amount = 0.00`，`fail_amount = 15.00`
  - `bill_status = 2`（失败）
  - `remark = "手动补发：1月有记录用户"`
