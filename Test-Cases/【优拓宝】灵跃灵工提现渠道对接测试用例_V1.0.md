# 【优拓宝】灵跃灵工提现渠道对接测试用例

## 基础信息

- **测试系统**: 优拓宝小程序（hcb-api）
- **数据库**: `hcb`
- **测试环境**: 192.168.1.60
- **日志路径**: `/home/tomcat/tomcat-hcbapi/logs/catalina.out`
- **代码分支**: test
- **接口列表**:
  - `com.hcb.checkWithdrawAccount` — 校验出金通道账户
  - `com.hcb.checkWithdrawalLimit` — 校验提现额度
  - `com.hcb.channelWithdrawSign` — 渠道用户提现签约
  - `com.hcb.withdrawApply` — 发起提现申请

## 1. 业务线路由测试

     ```bash
     ssh_execute: tail -n 100 /home/tomcat/tomcat-hcbapi/logs/catalina.out | grep "checkWithdrawAccount"
     ```

  - 日志中 `fenMiAppCode=YTB`（非 `RHX_TG`）
  - 使用优拓宝专属商户号，非任货行推广商户号

#### TC-002 channelWithdrawSign-优拓宝小程序-使用YTB业务线和小程序类型
- **前置条件**:
  - 系统: 优拓宝小程序
  - 数据库: hcb
  - 用户: 渠道用户，`IDCARD_URL` 包含正反面两张图（逗号分隔）
  - 测试账号: `b54a9f8a541740189725ce2a0b23ec3d`（林彬彬）
  - 环境: 192.168.1.60
- **测试步骤**:
  1. 用优拓宝小程序账号登录，进入提现签约页面

## 5. 提现申请测试

- **前置条件**:
  - 系统: 优拓宝小程序
  - 数据库: hcb
  - 配置: `sys_dictionaries.BIANMA=channel_withdraw_config`，`NAME_EN=100`（最低100元）
  - 用户: 渠道用户，余额充足
  - 环境: 192.168.1.60

     ```sql
     SELECT NAME_EN FROM hcb_sys_dictionaries WHERE BIANMA='channel_withdraw_config';
     ```

  3. 触发 `com.hcb.withdrawApply`
  4. 观察返回
- **预期结果**:
  - 返回 `ret=0`，提示「最低提现金额：100元起!」（根据配置动态展示）
