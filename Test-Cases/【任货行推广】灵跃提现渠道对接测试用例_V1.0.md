# 【任货行推广】灵跃提现渠道对接测试用例

## 测试基础信息

### 涉及系统
- **任货行推广** (hcb-api) - 发起提现
- **分米支付** (fenmi-pay) - 处理提现，调用灵跃API
- **灵跃平台** - 第三方提现/转账服务商

### 核心代码位置
- **fenmi代码**: `java/fenmi/fenmi-common/fenmi-common-bank/src/main/java/com/fenmi/common/bank/withdraw/`
  - `LingYueApi.java` - 灵跃API接口（第43行开始）
  - `LingYueAccountPaidImpl.java` - 提现实现类（第49行开始）
- **hcb-api代码**: 提现签约接口（增加身份证照片上传）

### 数据库表
- **fenmi_pay.withdraw_trade_order** - 提现交易订单表
- **fenmi_pay.withdraw_channel** - 提现渠道配置表
- **hcb.hcb_bank_withdraw_record** - 任货行提现记录表
- **MongoDB.bank_lingyue_log** - 灵跃接口日志表

### 接口文档
- **地址**: https://apidoc.lingyuejishu.com/71744001f0
- **签约模式**: 标签签约
