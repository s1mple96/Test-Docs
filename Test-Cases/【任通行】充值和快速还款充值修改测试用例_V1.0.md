# 【任通行】充值和快速还款充值修改测试用例

## 代码分析

### 1. 充值页面接口
- **接口路径**: `/etc/user/getRechargeTipsParam`
- **请求方式**: GET
- **请求参数**: `idCode`（可选，身份证号码）
- **Service实现**: `EtcCardUserServiceImpl.getRechargeTipsParam()`
- **代码位置**: `java/rtx/rtx-service/src/main/java/com/rtx/project/service/impl/EtcCardUserServiceImpl.java:1126-1180`

### 2. 接口返回字段
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
    "carCount": 16,           // 车辆数量
    "throughCount": 22,       // 30天内通行次数
    "proposalAmount": 3300.0, // 推荐充值金额（核心字段）
    "balace": -100.0,         // 钱包余额（注意：字段拼写错误，实际是balance）
    "totalBlackTimes": 93     // 账户总下黑次数
  }
}
```

### 3. 推荐充值金额计算逻辑
```java
// 推荐金额公式 = 账户欠款金额 + 推荐金额设置参数键值 X 车辆数量
String value = configService.selectConfigByKey("rtx.recharge.recommend.amount");
proposalAmount = new BigDecimal(value).multiply(BigDecimal.valueOf(carCount));

if (BigDecimalUtils.smallerThan(balace, BigDecimal.ZERO)) {
    // 账户余额为负数时增加欠款金额
    proposalAmount = proposalAmount.subtract(balace);
}
```

### 4. 配置参数
- **配置键**: `rtx.recharge.recommend.amount`
- **配置值**: 200元
- **公式**: 推荐金额 = 200 × 车辆数量 + 欠款金额（如果余额<0）

### 5. 数据库表
- **钱包表**: `rtx_user_wallet`
  - `balance`: 余额（负数=欠费）
  - `wallet_mode`: 钱包模式（1=1.0, 2=2.0）
  - `wallet_deduction`: 扣款模式
- **车辆用户表**: `rtx_etc_card_user`
  - `status`: 车辆状态
  - `user_wallet_id`: 关联钱包ID

## 需求说明

### 需求变更点

1. **快速还款入口（解黑页面）**
   - 欠费时：充值金额默认显示**欠款金额**
   - 欠款金额 = |钱包余额|（余额为负数时）

2. **余额充值入口（账户信息页面）**
   - 改成不支持仅充值欠款金额
   - 欠费时：充值金额默认显示**推荐金额**
   - 推荐金额 = 200 × 车辆数量 + 欠款金额

3. **去掉提示文案**
   - 删除"如需使用余额扣款,请返回首页点击[我的]-扣款设置-切换为余额扣款模式 进行切换"提示框

```sql
SELECT 
  uw.user_wallet_id,
  uw.id_code,
  uw.balance,
  uw.wallet_mode,

  200 * COUNT(ecu.id) as base_amount,
```

### 测试账号准备

```sql
-- 查询欠费账户（1.0余额模式）
SELECT 
  uw.user_wallet_id, 
  uw.id_code, 
  uw.balance, 
  uw.wallet_mode,
  ecu.car_num,
  ecu.status
FROM rtx_user_wallet uw
LEFT JOIN rtx_etc_card_user ecu ON uw.user_wallet_id = ecu.user_wallet_id
WHERE uw.balance < 0 
  AND uw.wallet_mode = '1'
  AND ecu.status IN ('0', '2')
LIMIT 10;

-- 查询正常余额账户（1.0余额模式）
SELECT 
  uw.user_wallet_id, 
  uw.id_code, 
  uw.balance, 
  uw.wallet_mode,
  ecu.car_num,
  ecu.status
FROM rtx_user_wallet uw
LEFT JOIN rtx_etc_card_user ecu ON uw.user_wallet_id = ecu.user_wallet_id
WHERE uw.balance >= 0 
  AND uw.wallet_mode = '1'
  AND ecu.status IN ('0', '2')
LIMIT 10;

-- 查询多车辆账户
SELECT 
  uw.user_wallet_id, 
  uw.id_code, 
  uw.balance, 
  COUNT(ecu.id) as car_count

LEFT JOIN rtx_etc_card_user ecu ON uw.user_wallet_id = ecu.user_wallet_id
WHERE ecu.status IN ('0', '2')
GROUP BY uw.user_wallet_id
HAVING car_count >= 3
LIMIT 10;
```
