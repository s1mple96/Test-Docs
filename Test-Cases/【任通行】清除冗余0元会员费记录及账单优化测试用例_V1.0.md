- **前置条件**: 
  - 系统: 任通行小程序
  - 数据库: rtx

  - 测试环境: 192.168.1.60
- **测试步骤**:
  1. 登录任通行小程序。
  2. 进入“我的” -> “我的权益” -> 点击权益卡 -> “权益账单”。
  3. 观察账单列表中的记录。
- **预期结果**: 
  1. 账单列表中仅显示金额大于 0 的记录。
