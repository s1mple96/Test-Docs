# 【任货行】月管理费和先享月费非湖南江苏省份迭代测试用例

> 版本：V1.1（2026-03-23）
> 关联提交：89c5ed6（added：增加不扣月管理费用户先享月费账单生成）

---

## 需求说明

**一、月管理费扣款（Fee_Silent_Wallet）新增逻辑**
- 有配置省份（Fee_Silent_Wallet 的 task_require 中有 operatorCode key 的运营商，如TXB、MTK），在按原逻辑扣款的同时，**额外插入一条**：
  - `bill_type = 3`（月管理费账单2）
  - `bill_amount = 15元`
  - `bill_status = 2`（扣款失败）
  - `fail_amount = 0`
- 无配置省份（如XTK、LTK、HTK等），不插入月管理费账单2，走原有流程

**二、全量钱包账单表新增账单类型筛选**
- 账单类型下拉新增"月管理费账单2"选项（bill_type=3）

**三、先享月费账单生成改用 Fee_Silent_Wallet 配置**
- **有配置运营商**（TXB、MTK等，jsonObject 中含 operatorCode key）：查询 `bill_type=3`（月管理费账单2）的失败账单汇总，生成先享月费账单
- **无配置运营商**（XTK、LTK等，jsonObject 中无 operatorCode key）：查询 `bill_type=1`（月管理费账单）的失败账单汇总，生成先享月费账单
- 开发备注：起诉用户统一都不扣先享月费

**四、数据初始化（新增需求）**
- 通过接口 `/manualGenerateAndDeductBills` 一键完成以下流程：
  1. 补录1月份 bill_type=3：对 LTK/HTK/JTK/MTK/CTK 1月份有 bill_type=1 的**用户**（按用户聚合），各插入一条 `bill_type=3, bill_amount=15, bill_status=2, bill_date=20260101`
  2. 补录2月份 bill_type=3：同上，`bill_date=20260201`
  3. 触发先享月费账单生成（汇总1-2月份 bill_type=3 数据）
  4. 触发先享月费账单扣款
  5. 补录3月份 bill_type=3：同上，`bill_date=20260301`（仅补录，不触发生成和扣款）
- 覆盖运营商：LTK、HTK、JTK、MTK、CTK（不含湖南 XTK，不含江苏 TXB）

**当前数据库配置**（task_require = `Silent_Wallet&{"TXB":30,"MTK":20}`）：
- **有配置（收月管理费）**：TXB、MTK
- **无配置（不收月管理费）**：XTK、LTK、CTK、JTK、HTK 等
- **本次初始化范围**：MTK（非湖南、非江苏，有配置）及其他非XTK/非TXB运营商
