## 3. 统计计算正确性验证

### 3.1 扣减计算验证

#### TC-017 验证统计表通行服务费 = 收入 - 退费
- **前置条件**：
  - 已知某月（如2026-03）通行服务费收入总额（可从原有统计数据查看）
  - 已知该月通行服务费退费总额（从明细列表合计或底部合计行获取）
- **测试步骤**：
  1. 进入明细页，查询 2026-03 **通行服务费收入** 明细，记录底部合计金额（收入合计）
  2. 查询 2026-03 **通行服务费退费** 明细，记录底部合计金额（退费合计）
  3. 手动计算：预期净值 = 收入合计 - 退费合计
  4. 进入统计表，查询 2026-03
  5. 找到任货行-通行服务费行，查看"含税合计"或"价税合计"
- **预期结果**：统计表中的金额 = 收入合计 - 退费合计（误差 ≤ 0.01元）

#### TC-018 验证退费金额为0时统计结果不受影响
- **前置条件**：找到一个只有通行服务费收入、没有退费的月份
- **测试步骤**：
  1. 确认该月退费明细为空（TC-012方式）
  2. 查看该月统计表的通行服务费金额
  3. 与收入明细合计对比
- **预期结果**：无退费时，统计金额 = 纯收入金额，不影响原有数据

#### TC-019 验证统计表中退费项不单独显示行（统计已合并）
- **前置条件**：2026-01 有退费数据
- **测试步骤**：
  1. 进入统计表，查询 2026-01
  2. 查看任货行相关行，确认展示方式
- **预期结果**：
  - 统计表中通行服务费只有一行（净值）
  - 无单独的"通行服务费退费"行

### 3.2 税率计算

#### TC-020 验证统计表中通行服务费的税率字段
- **前置条件**：2026-01 有退费数据后的统计表
- **测试步骤**：
  1. 进入统计表查询 2026-01
  2. 找到任货行-通行服务费行，查看税率、不含税、税金、价税合计
- **预期结果**：
  - 税率 = 6%（固定税率）
  - 不含税金额 = 含税金额 / 1.06（保留2位小数）
  - 税金 = 含税金额 - 不含税金额
  - 价税合计 = 不含税金额 + 税金（即含税金额）

## 附录：数据核查SQL参考

> 以下SQL用于辅助测试验证，在财务库中执行（需确认具体库名）。

```sql
-- 查询26年1月通行服务费退费明细记录数
SELECT COUNT(*) FROM fn_item_detail
WHERE item_detail_type = 2007
  AND operator_type = 20070002
  AND item_date = 202601
LIMIT 1;

-- 查询26年1-3月退费金额合计（按月分组）
SELECT item_date, SUM(amount) refund_total
FROM fn_item_detail
WHERE item_detail_type = 2007
  AND operator_type = 20070002
  AND item_date IN (202601, 202602, 202603)
GROUP BY item_date;

-- 查询统计表中通行服务费数据（需确认count_date格式）
SELECT * FROM fn_item_statistics
WHERE item_detail_type = 2007
  AND count_date IN ('2026-01', '2026-02', '2026-03');
```

> **说明**：
> - `item_detail_type = 2007` 对应 `RHX_DEDUCE_SERVICE_FEE`（货车通行服务费）
> - `operator_type = 20070001` = 通行服务费收入；`20070002` = 通行服务费退费
> - 具体数据库连接信息需从财务服务配置（Nacos）中获取
