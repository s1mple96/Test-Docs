# 【任货行】清欠用户快速还款测试用例

## 1. 查询页面功能测试

> **重要说明**: 查询页面有4个校验条件，必须**同时满足**才能进入账单页面：
> 1. ✅ 用户状态 = 正常(STATUS=1) 或 ETC黑名单(STATUS=2)
> 2. ✅ 钱包标签 = 有PURGE_Wallet标签
> 3. ✅ 钱包余额 < 0
> 4. ✅ 冻结金额 < 钱包余额绝对值
>
> **校验顺序**: 按上述顺序依次校验，任一条件不满足则拦截。

### 2.2 金额计算公式验证

> **公式说明**:
> 
> **符号定义**:
> - D = |钱包余额| (钱包余额绝对值)
> - F = 冻结金额 (过往留存金额)
> - T = 待冻结金额 (优惠减免金额)
> - r = 0.6% = 0.006 (微信手续费率，从配置读取)
> - Q = D - F - T (实际需还金额)
> - S = Q × 0.6% = Q × 0.006 (充值手续费)
> - P = 最后还款金额 (用户实际支付)
>
> **完整公式推导**:
> ```
> 1. 实际需还金额:
>    Q = 钱包余额绝对值 - 冻结金额 - 待冻结金额
>    Q = D - F - T
>
> 2. 充值手续费:
>    S = 实际需还金额 × 0.6%
>    S = Q × 0.6%
>    S = Q × 0.006
>
> 3. 总欠款 (页面展示):
>    总欠款 = 钱包余额绝对值 + 充值手续费
>    总欠款 = D + S
>
> 4. 最后还款金额 (用户支付):
>    P = 实际需还金额 + 充值手续费
>    P = Q + S
>    P = Q + Q × 0.006
>    P = Q × (1 + 0.006)
>    P = Q × 1.006
>    (向上取整到分)
>
> 验证: P = 总欠款 - F - T = (D + S) - F - T = D - F - T + S = Q + S ✅
> ```
>
> **关键公式**: 
> - ✅ P = Q × 1.006 (因为 1.006 = 1 + 0.006 = 1 + 0.6%)
> - ❌ P = Q / 0.994 (错误公式)
