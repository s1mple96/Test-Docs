# 【任货行】清欠用户快速还款新增京东白条支付测试用例

> 需求：任货行清欠用户快速还款原来直接调用微信支付，现在改为拉起收银台，让用户选择微信支付或京东白条支付两种方式。需要校验充值记录、统一交易、钱包流水的支付方式数据，以及手续费配置和实际收款方。

---

## 1. 收银台展示测试

### 1.1 收银台入口

#### TC-001 清欠用户-快速还款-拉起收银台-展示微信支付和京东白条两种支付方式
- **前置条件**:
  - 系统: 任货行APP（小程序）
  - 数据库: hcb
  - 场景: 用户钱包余额为负（欠款状态），进入清欠快速还款入口
  - 测试环境: 192.168.1.60
- **测试步骤**:
  1. 登录任货行小程序，找到已有欠款的测试用户
  2. 进入清欠还款页面
  3. 点击「快速还款」按钮
  4. 观察是否弹出收银台
  5. 检查收银台中的支付方式列表
- **预期结果**:
  - 弹出收银台页面
  - 收银台展示两种支付方式：**微信支付** 和 **京东白条支付**
  - 两种支付方式均可点击选择

#### TC-002 清欠用户-快速还款-收银台-展示还款金额
- **前置条件**:
  - 系统: 任货行APP
  - 场景: 用户有欠款，进入快速还款收银台
- **测试步骤**:
  1. 进入快速还款，拉起收银台
  2. 检查收银台显示的还款金额是否正确（与欠款金额一致）
  3. 核对数据库欠款金额：查询 `hcb.hcb_truckuserwallet` 用户钱包余额（负值）
- **预期结果**: 收银台显示的还款金额与用户欠款金额一致

## 2. 微信支付方式测试

### 2.1 微信支付还款

#### TC-003 清欠用户-快速还款-选择微信支付-还款成功
- **前置条件**:
  - 系统: 任货行APP
  - 数据库: hcb
  - 场景: 用户有欠款，拉起收银台
  - 已授权微信支付
- **测试步骤**:
  1. 拉起收银台，选择「微信支付」
  2. 完成微信支付流程（输入支付密码）
  3. 支付成功后，查询充值记录：`SELECT * FROM hcb.hcb_topuprecord WHERE TRUCKUSER_ID=? ORDER BY CREATE_TIME DESC LIMIT 1`
  4. 核查 `PAY_WAY` 字段是否为微信支付对应的值
  5. 核查 `actual_payee` 字段
  6. 查询钱包余额变化：`SELECT * FROM hcb.hcb_truckuserwallet WHERE id=?`
- **预期结果**:
  - 还款成功，页面提示成功
  - `hcb_topuprecord.PAY_WAY` 为微信支付对应值（0）
  - `actual_payee` 为分米（FM）
  - 钱包余额恢复正常（欠款清零或减少）
  - 钱包冻结金额正确解冻

#### TC-004 清欠用户-快速还款-选择微信支付-查询充值记录支付方式准确
- **前置条件**:
  - 已完成 TC-003，微信支付还款成功
- **测试步骤**:
  1. 登录 hcb-admin 管理后台
  2. 查询该用户的充值记录
  3. 确认支付方式显示为「微信支付」
  4. 查询统一交易记录，确认支付方式字段
- **预期结果**: 充值记录和统一交易记录中，支付方式均显示为微信支付，数据准确

#### TC-005 清欠用户-快速还款-选择微信支付-查询钱包流水支付方式准确
- **前置条件**:
  - 已完成 TC-003，微信支付还款成功
- **测试步骤**:
  1. 查询 hcb 数据库钱包流水记录（`hcb_etccard_balance_flow` 或相关表）
  2. 找到对应的还款记录
  3. 确认流水的支付方式字段
- **预期结果**: 钱包流水中支付方式准确，标注为微信支付

- **前置条件**:

  - 数据库: hcb、fenmi_pay
  - 场景: 用户有欠款，拉起收银台
  - 分米支付: fenmi_pay 中已配置 JING_DONG_QUICK_PAY（或 YUN_ZHUO_JD_BT_PAY）通道，status=1

  5. 核查 `actual_payee` 字段是否为分米（FM）
  6. 查询钱包余额是否恢复
  7. 查看日志：`tail -n 100 /home/tomcat/tomcat-hcbapi/logs/catalina.out`
- **预期结果**:

#### TC-007 清欠用户-快速还款-选择京东白条支付-充值记录手续费准确
- **前置条件**:
  - 已完成 TC-006，京东白条还款成功
  - 京东白条手续费与微信支付手续费单独配置（不同费率）

#### TC-009 清欠用户-快速还款-选择京东白条支付-查询钱包流水支付方式准确
